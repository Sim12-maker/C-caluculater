# C++ caluculater
A straightforward C++ calculator that performs

1.Arithmetic Unit{"+","-","*","/"}  

2.Shapes areas{Square,Triangle,Rectangle,Circle-radius}

=======⚙️ FEATURES =========

2.Supports repeated calculations in a loop.

3.Colored output

4.Validates user input and displays error messages.
